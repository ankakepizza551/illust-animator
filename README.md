# illust.animator

1枚のイラストに、命を吹き込む。AIを活用したイラストアニメーションツール集。

## 🚀 収録ツール

### 1. AI Part Animator (`hair-animator.html`)
AI（Gemini API）を活用してイラストの部位を自動検出し、アニメーションを付与します。
- **AI部位検出**: 髪の毛、スカート、リボン、袖などを自動的にポリゴンで囲みます。
- **多彩なアニメーション**: ゆらゆら、なびく、はずむ、波打つなど。
- **インタラクティブ機能**: マウスや指で触れてイラストを揺らすことができます。
- **背景自動補完 (β)**: 動かした後の隙間を埋めるインペイント機能。
- **エクスポート**: GIF, WebM, 透過WebP, APNGに対応。

### 2. Basic Animator (`animator.html`)
イラスト全体にエフェクトや動きをつける軽量なアニメーションツール。
- **全体モーション**: ふわふわ、呼吸、脈打ちなど。
- **視覚演出**: グロー効果、パーティクル演出。

## 🛠 使い方

1. `index.html` をブラウザで開きます。
2. 使用したいツールを選択します。
3. イラストをドラッグ＆ドロップします。
4. （AI Part Animatorの場合）Gemini APIキーを入力して「部位を自動検出」を押します。
5. 設定を調整して、お好みの形式で保存します。

## 💻 開発・実行環境

- HTML / Vanilla JS / CSS
- [Gemini API](https://aistudio.google.com/) (部位検出に使用)
